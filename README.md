# Clarity

Clarity is a powerful, beautifully designed Desktop Application for File Organization, PC Optimization, and System Cleaning. Built with Electron, React, and Vite, it offers an incredibly smooth user experience with support for multiple languages and modern aesthetics.

## 🚀 Features
- **File Sorter:** Intelligently organize messy directories by grouping files into categorized folders based on extensions.
- **RAM Optimizer:** Visualize your active memory load and purge inactive working sets to free up system resources.
- **System Sweeper:** Quickly detect and wipe temporary files, Windows cache, and junk data.
- **Double-Take Finder:** Scan and safely remove duplicate files to recover precious disk space.
- **Empty Folder Purger:** Recursively locate and remove empty, useless directories.
- **Big File Radar:** Scan the filesystem to locate and optionally delete the heaviest files hogging storage.
- **Clipboard Vault:** Securely keep a history of copied text fragments for quick access.
- **Startup Optimizer:** Manage applications that launch on Windows boot. Enable or disable startup items with a click.

## 🛠 Installation

### For Regular Users (Windows)
1. Go to the **Releases** page (or use the provided `.exe` file).
2. Download `Clarity-Setup.exe`.
3. Run the installer.
4. During setup, **you can choose the installation folder** of your preference.
5. Launch Clarity from the desktop shortcut or start menu!

### For Developers
If you want to run Clarity from the source code or build it yourself:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/cvzxeov/CLARITY
   cd clarity
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in Development Mode:**
   ```bash
   npm run dev
   ```

4. **Build the `.exe` Installer:**
   ```bash
   npm run build
   ```
   *The final `.exe` installer will be located in the `release/` folder. The installer allows custom path selection (NSIS configuration).*
